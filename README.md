## Getting started
This project requires [trunk](https://crates.io/crates/trunk) & [yarn]();
Uses [bonnie]() to do command aliases:
### Installation
Install bonnie
```console
cargo install bonnie
```

Install all dependencies:
```console
bonnie install
```
### Development
Run development server:
```console
bonnie serve
```
build release:

```console
bonnie build
```
